# 开发规范

## 语言与沟通
- **输出**: 所有 AI Agent 输出必须使用 **中文**。
- **代码注释**: 所有代码注释必须使用 **中文**。

## 数据库
- **原则**: **严禁删除任何数据库内容**。
- **操作**: 仅在确信安全的情况下进行数据修改或添加。

## 代码质量
- **类型安全**: 严格遵守 TypeScript 类型定义，避免 ny。
- **模块化**: 保持代码高内聚低耦合。

## 代码质量
-**国际化支持，开发的任何一个功能都以翻译键的方式存在（不可硬编码语言）

## 背景颜色
-***开发的功能，不能硬编码背景，必须支持深色和浅色
